Project Details

ConGo E-commerce is a Django-based project that amalgamates Python, HTML, CSS, and MySQL. It delivers an intuitive and secure online shopping platform with features like product browsing, a user-friendly cart, and robust user authentication. By utilizing MySQL, it efficiently manages data while presenting an aesthetically pleasing and responsive web interface.
